Notre application permet de afficher les restaurants d'après une api donnée, donc elle permet d'afficher, supprimer et ajouter un restaurant. Ansi que la recherche du restaurant. 
Pour chaque restaurant vous pourrez consulter ses données dans une page séparer, ansi que sa localisation dans google map. en plus dans une autre page vous aurez la possibilité de consultez la carte et les menu  de restaurant, notre site web permettre générer les données du carte et menu d'une manier aléatoire d'auprès une une list statique. Finalement vous pouvez commander des plat et des menu juste par une clicker, vous pouvez également consulter ou supprimer Les plats choisies. Le prix total s'affichera pour vous montrez le montant a payer.


Remarque : nous avons changé dans le server la method qui permette d'insérer dans la base de données  pour qu'on puisse insérer la ville de restaurant aussi , donc je vous invite a telecharger notre version du server (inclus dans le repertoire git hub )
